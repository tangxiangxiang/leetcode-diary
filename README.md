# Leetcode Diary

📒 **Leetcode Diary** 是一个基于 [VitePress](https://vitepress.dev/zh/) 构建的个人学习项目，用于记录和分享 LeetCode
刷题笔记与算法学习心得。

通过 GitHub Actions 自动部署到 GitHub Pages，让你的刷题笔记随时随地可访问。

---

## 🌟 功能特性

- 使用 VitePress 提供简洁高效的文档界面。
- 记录和分类 LeetCode 题解。
- 支持代码高亮、数学公式和 Markdown 格式。
- 自动化部署到 GitHub Pages。
